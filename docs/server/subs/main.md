## 简介
Comet 生存服务器是以生存和玩家互动为核心的服务器，加入这个服务器，玩家将出生在位于地下 y 轴 -52 的深黑古城。玩家需要利用求生的智慧，与其他玩家一同逃出困境、搜寻物资，最终踏入生存之道。

## 游戏规则
- 游戏难度：困难
- **出生点保护：出生点建筑范围内为冒险模式，范围外为生存模式**
- **死亡不掉落**
- 火焰蔓延正常
- 生物破坏正常开启，不影响村民繁殖和末影人搬方块
- 有25%的在线玩家睡觉即可跳过夜晚
- **可获得 Xbox 成就**
- 玩家定位条开启

- 默认发放初始装备
- 使用`追溯指针`打开服务器菜单
- **领地系统**
- 经济系统与通用货币系统
- 耕地无法被踩坏
- 树叶快速消失
- 苦力怕防爆，TNT和其他生物无防爆
- 下界TNT爆炸不会产生火焰
- 死亡会掉落一定数额的金币

## 被偷家的应付策略
当你发现自己的基地被掠夺，应采取一系列防护或报复措施。以下是应对建议：
- **手动查找玩家坐标**：使用`/findplayer`指令，可对在线玩家进行定位。关于该功能的详情请见<a href="/plugins/findplayer" target="_blank">寻仇系统</a>。
- **记录当时在线玩家**：当您怀疑自己的基地是被当时的在线玩家破坏时且情节恶劣，可记录玩家列表以保留证据，必要时可一一询问。
- **向管理员寻求帮助**：当发现基地被严重破坏且自己无力挽回后，可以联系管理员。在人道关怀的原则下，管理员会提供一些基础物资帮助。
