# Comet 枪战服务器 - 特色游戏玩法介绍
## 标签注释
- <Badge type="info" text="原创" />：由参加服务器建设的 Comet 玩家和 Comet 项目组成员亲自规划与打造的地图或玩法，某些地图的灵感可能来自于周边事物或是某个游戏。
- <Badge type="info" text="非经典" />：指某个游戏玩法或某个地图建议的玩法并不是枪战玩法，而是更倾向于其他玩法。遵循该玩法进行游戏，可获得更好的游戏体验。（不强制玩家的武器，但是很建议玩家在该种玩法下仅使用推荐的武器）

## 简介
特色游戏玩法是我们在经典枪战玩法上做出的一些创新，皆在为玩家们提供独特、难忘的游戏体验。这些游戏模式通常配备在一些具备经典枪战玩法的地图上，供玩家们手动开启，或配合专门地图出现。

## 游戏玩法介绍
### 跳狙飞人
- 实装日期：2023.8.19
- 实装地图：遗迹
#### 设定
1. 该模式被激活后，在正式开始游戏之前，系统会为所有参赛的玩家发放一把狙击枪和若干弹匣。
2. 在游戏过程中，玩家始终有缓降和跳跃提升效果。

#### 注意事项
1. 玩家使用狙击枪(AWM)参与战斗可获得最佳的游戏体验。
2. 系统向玩家发放的狙击枪会在游戏结束后统一扣除，不论玩家输赢情况。玩家自己购买的狙击枪我们不做限制。
3. 在游戏过程中，请注意区分发放的物品和自己的物品，不要把发放的物品带出地图，若在场外遇到吞物品现象，其造成的后果由玩家自负。

### 无限火力
- 实装日期：2024.2.16
- 实装地图：星球
#### 设定
该模式被激活后，在正式开始游戏之前，系统会为所有参赛的玩家发放一把枪和一个与之配套的，但是用不完的弹匣。
#### 注意事项
1. 系统向玩家发放的物品会在游戏结束后统一扣除，不论玩家输赢情况。玩家自己购买的相关物品我们不做限制。
2. 在游戏过程中，请注意区分发放的物品和自己的物品，不要把发放的物品带出地图，若在场外遇到吞物品现象，其造成的后果由玩家自负。

### 捉迷藏   <Badge type="info" text="非经典" />
- 灵感来自：《Among Us》
- 简介：该玩法借鉴于游戏《Among Us》中的捉迷藏模式。在该游戏模式中，玩家分为两部分：躲藏者与追杀者。追杀者的任务是消灭地图中所有的躲藏者，而躲藏者在游戏中需要四处逃跑以躲过追杀者的追捕。

#### 设定
1. 单局限时3分30秒，分为玩家躲藏时间（3分钟）和尾杀时间（30秒）；不支持更改游戏时间。
2. 单局游戏仅存在一名追杀者；单局游戏最多人数为10人。
3. 在游戏中，所有玩家视野有限；追杀者的视野大于躲藏者。
4. 所有玩家武器不限。追杀者无法被杀死（无敌）；躲藏者之间可以互相伤害（但不建议这么做）。
5. 游戏场景中设有任务点。玩家每在一个任务点停留3秒钟就能让游戏时间缩短3秒；每个任务点单局只能使用一次。
6. 玩家躲藏时间过后会进入30秒的尾杀时间。在该时间段下，所有任务点将会被停用，同时追杀者会被强化，即移动速度会略微增加，但视野不会变。

#### 几点建议
1. 建议追杀者不使用远程武器，仅使用近战武器进行游戏。（在分配队伍时系统会为追杀者发放一把近战武器）
2. 躲藏者之间尽量不互相造成伤害。

:::info 
该游戏模式暂未上线。
:::
