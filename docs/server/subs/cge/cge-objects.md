# Comet 枪战服务器 - 服务器物资详情
## 关于免费发放物资
商店中的“免费发放物资”板块提供一些基础武器和防具，玩家无需花费任何资金就能取得。

## 枪械
枪械是玩家在这个服务器中最常用的武器。

枪械需要弹匣作为弹药。对于不同的枪械，通常都具有特定的弹匣，而且对于枪械来说，其弹匣提供的容量也是固定的。

在开枪之前请确保您的背包有足够的弹药，并且和您武器的型号对应。在物资商店中，枪械与弹药的排列通常一一对应，以保证您不会买错。

部分枪械支持更改枪械配件，详情另见其他小节。

#### 关于弹匣的特殊点：
对于霰弹枪（勃朗宁 725和雷明顿 870 ），它们统一使用猎枪弹作为弹药。

对于RPK枪支，还有对应的弹鼓可以使用。弹鼓可以直接充当普通弹匣使用，并且可以为枪械提供额外的子弹容量。

#### 关于枪械常见的使用方法：
查看子弹剩余量：在HUD的右下角可以看到所持枪械的剩余子弹数量。对于移动端玩家，您需要前往设置将UI设置为“经典”才能看到剩余量。

开枪：长按屏幕或单击鼠标右键可以发射子弹。对于移动端玩家，使用摇杆控制模式（非方向键）可能会更方便开枪。

换弹：若您的枪械弹夹已经用尽且背包还有对应型号的弹匣，可以自动换弹。在枪械弹夹不满且未潜行的情况下，点击屏幕或单击鼠标左键可以手动换弹。

瞄准或狙击：手持枪械保持潜行动作即可瞄准，对于任何枪械都适用。
如果您手持的是狙击枪并且配备默认镜头，则您会进入狙击状态，在这种状态下您的视角缩放的更远，但视野会受限。

在瞄准状态下您会射击得更准，但行动速度会降低。

## 防具
防具可以使玩家抵抗住更多的伤害，增加在战斗中的存活几率。

在战斗之前系统会给予初始护甲用于分清敌我阵营。每一个地图中的初始护甲可能都互不相同。有些地图会提供特殊材质的护甲。

商店也提供若干种类齐全、属性丰富的护甲，并使用不同颜色标注物品的总体属性。

由于实现原因，目前所有护甲均为一次性，即在每局游戏结束后、进入游戏时、分配队伍时，玩家身上的盔甲会被清空。

头盔是目前分辨敌我队伍的唯一标识。所有地图的默认盔甲除头盔均支持卸下，同时商店中售卖的所有盔甲也能够卸下。

#### 对盔甲而言，不同颜色代表的属性见下文：
- 白色（淡灰色）：劣质物资，通常无任何附魔，甚至有些买回来到手耐久值都不是完整的。
- 绿色：基础物资，通常都具有保护1等附魔。
- 蓝色：中等物资，在绿色物资的基础上，有经验修补附魔和更强的保护附魔。个别护甲有专门保护（比如摔落保护等）的附魔。
- 紫色：高端物资，在蓝色物资的基础上，有迅捷潜行附魔和更强的保护附魔。个别护甲有多个专门保护（比如火焰保护等）的附魔，并且和普通保护附魔共存。
- 橙色：极品物资，具备几乎涵盖各方面的专门保护附魔，并且和普通保护附魔共存。
- 红色：传奇物资，几乎具有以上所有品质具备的属性，但个别物资会给予玩家debuff（比如缓慢）。

## 冷兵器
冷兵器在这个服务器中也是非常重要的存在。

冷兵器包括近战武器（剑、工具）、三叉戟、弓和弩。

物资商店中提供各种各样的冷兵器，每一个冷兵器都具备不同的属性和附魔。

个别冷兵器在玩家手持的时候会给予增益buff。

地图中也会提供冷兵器。通常存在于地图的容器之中，并且会在游戏开始时刷新或定时刷新。

## 辅助物资
商店提供一些辅助物资，包括辅助武器、药水、辅助物品等。

辅助武器包括燃烧弹、手榴弹等物品。玩家手持燃烧弹长按屏幕或单击鼠标右键可以将其发射，被命中的目标会着火，且发射路径呈直线型，不受重力影响。

药水包括增益药水与反向增益药水，包括但不限于生命恢复、夜视、抗火和中毒。

辅助物品包括望远镜、护甲修补剂、除颤仪等。除颤仪的效果和不死图腾一致；护甲修补剂可以用在有经验修补附魔的盔甲上。

## 专武
有些武器是物资商店中买不到的，我们称之为专武。

专武涵盖各类武器，包括枪械、冷兵器等。

专武一般在游戏地图中提供。您可能会在部分地图的特色游戏玩法中获得专武，或是在地图的某个容器中获得。

通常您无法将专武带出游戏地图。一旦服务器检测到您在规定的游戏地图外手持这类规定的物品或与之配套的物品，服务器会立马清除该物品（不是整个背包）。

#### 专武详情：
遗迹地图：特色游戏模式下服务器会为参与游戏的玩家提供狙击枪和一些弹匣，这些物资仅可以在当局中使用。

星球地图：特色游戏模式下服务器会为参与游戏的玩家提供特种枪（RA MX115）和无限弹匣，这些物资仅可以在当局中使用。该枪支不在商店中售卖。

神庙地图：在开始游戏的时候地图中央的箱子会生成一把秒杀剑，该物品可以带出地图。

## 语音包
服务器提供一些特色语音包，呈物品形式，玩家可以在对局中使用它，以增加游戏的乐趣。

语音包若被玩家正确装载，在规定的某个时间戳会向所有在线玩家播放一段语音并扣除该玩家物品。（该物品为一次性）

#### 使用方法：
将语音包放在物品栏最后的位置（不包括快捷栏），即可正确装载。

## 枪械配件
枪械配件可以强化枪械，增加枪械的属性，例如射击精度、射击距离等等。
枪械配件目前包括若干镜头、前握把、榴弹发射器、弹鼓等。

并非所有的枪械都支持添加枪械配件。

也并非所有的配件都支持任意摆放。例如弹鼓是充当普通弹匣使用的，你仍然需要把它放在规定的物品栏上才能使用。

狙击枪也支持更换镜头。若狙击枪更换了其他镜头，其在瞄准的状态下缩放状态和普通枪支无异。

有些特别的枪械还集成了额外的枪械配件，玩家可随意启用或关闭。例如RPK系列枪支，玩家点击屏幕或单击鼠标左键可以开关枪械支架。

#### 使用方法：
对于镜头：将镜头放在快捷栏第9格（从右往左数第一格）即可生效。配件和枪械不为一体。

对于弹鼓：将弹鼓放在快捷栏第8格（从右往左数第二格），在弹匣用尽的情况下换弹即安装成功。

## 特种榴弹
特种榴弹是适用于枪械配件——“M203榴弹发射器”的新型弹药，包括脏弹、烟幕弹、五彩弹等，不包括HE弹（高爆榴弹）。

特种榴弹需要搭配“M203榴弹发射器”这个枪械配件，而不是M79。M79只能填装HE弹（也叫做高爆榴弹）。

#### 使用方法：
将M203榴弹发射器装载到支持的枪械上（放在副手位置），将特种榴弹放在快捷栏第8格（从右往左数第二格），然后保持潜行状态，点击屏幕或单击鼠标左键进行换弹操作，等待装填完成即可发射。
