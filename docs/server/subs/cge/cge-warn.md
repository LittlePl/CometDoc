# Comet 枪战服务器 - 防无聊忠告
#### to 仅冲着枪战服务器来的玩家：

我们枪战服的众多特色，包括高质量枪械、多张原创地图、休闲区、高度定制化菜单、一些彩蛋，以及丰富的音乐（音乐盒、音乐盛典等系统。每个地图也有独特的音乐，通常都会在游戏结束前3分钟自动播放）、生动的语音播报等，始终欢迎任何人来随时体验。

Comet 枪战服务器可能会随游戏版本更新而出现无法运行的情况，这通常是因为服务器所使用的枪械模组暂未适配新版（具体表现为部分枪械无法对实体造成伤害），或者平时（非寒暑假期间）玩枪战服的人不是特别多。再者，此服务器的运行会增加其他服务器的运行负担（可能会造成卡顿），故我们视情况将该服务器关闭，等到玩家主动要求或人员高峰再启动。

这边推荐您玩一玩我们的其他服务器（生存服务器、空岛服务器），都很优质，玩家氛围很好，玩家质量很高，社区化很强，插件定制化特别高，防熊服措施做的特别到位，生存服还可以获取Xbox成就。枪战服务器需要至少2个玩家才能玩，而且一般仅仅因为枪战服而来到我们服务器的人，大多数都不会在服务器特别活跃、待的特别久，久而久之就无聊了，然后成潜水玩家了。所以我们在宣传中对该服务器描述的不是很细致。

我们的服务器定位主要是原版生存和空岛，小游戏次之，仅为玩家们提供即时放松的场所和对服务器起到宣传辅助的作用。

我们欢迎每一个追求和谐友爱的玩家来到我们的服务器，建立社区、与其他玩家共同发展。
