# Comet 枪战服务器 - 广播播音员溯源
## 简介
在 Comet 枪战服务器中，我们设立仿生广播功能，除了文字提醒外，我们还使用了来自各个动漫或游戏角色的声音去引导玩家游戏，这无疑激发了玩家的游戏兴趣并极大地丰富了玩家的游戏体验，可谓生动形象、独具创新、极具韵味。但一定要注意的是，在公共场合玩这个服务器的时候，不要将设备音量调的太高，否则有可能会让你当场社死 ()。

我们在此列举了所有地图广播的人物全名以及人物来源，感兴趣的玩家可以去翻阅相关维基。

## 详情
:::info
1. 我们让广播的格式与玩家聊天保持一致，以减少玩家割裂感并维持玩家游戏体验，但所有来自广播的消息我们会使用黄色文字。请玩家注意区分广播和来自真实玩家的消息，不要混淆。
2. 为了实现广播信息和聊天信息格式一致，所有角色的名字在聊天栏中显示的是其英文名。
3. 所有广播的声音均为语音模型推理而成，没有使用任何真人真声。
:::
