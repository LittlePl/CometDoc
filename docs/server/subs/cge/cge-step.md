# Comet 枪战服务器 - 基本游戏流程
如果您刚来到这个服务器，不了解这个服务器的基本玩法，您可以看一下这里，我们制作了一个分步流程供您参考。

1. **完成每日签到**，你将获得一定的资金。

2. 刚来到服务器您的物品栏里一定有个**时钟**吧，这个东西可以**打开服务器菜单**。（手持时钟点击或右键单击方块即可打开服务器菜单）

3. 通过服务器菜单**访问物资商店**。这将是你以后游玩这个服务器的重要操作。

4. **购买一些物资用于战斗**。更昂贵的物资在您有了足够的资金之后才能购买，并且注意在购买枪械的时候不要忘记购买对应的弹匣，物资商店的排列顺序通常一一对应，以保证您不会买错。

5. 在完成以上操作后即代表您已经准备好了。走出出生点前方的大门，您会看到几个按钮，它们可以更改部分游戏设置，也可以让你开始游戏。**点击开始游戏的按钮**后，进入游戏的入口会亮起，这时您就可以**走向游戏入口然后稍等待一会**，您会自动进入游戏。

最后在作战中大显身手吧！
