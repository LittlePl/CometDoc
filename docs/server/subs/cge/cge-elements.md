# Comet 枪战服务器 - 基本游戏元素
## 资源包类
- **枪械资源包**：我们使用《Radium's Armament》作为本服务器的主要资源包，它为服务器提供了极其重要的枪械元素和场景元素。
:::warning 声明
服务器仅供玩家体验该资源包，杜绝一切抄袭与外传。对这个资源包感兴趣的玩家可以自行去上网了解。
:::
- **其他资源包**：除了枪械资源包，我们还装载了家具模组以丰富游戏场景，并且自主制作了为服务器提供音频和图片的资源包，该资源包内的文件可以被游戏调用，您在游戏过程中之所以能听到动听的音乐，是因为有着它们的支持。

## 建筑类
- **地图**：玩家们作战的地方。服务器配备丰富的地图供玩家挑选，每个地图具备的游戏设置可能不同。同一时间在线玩家只能游玩一张地图。
- **场景**：为玩家们提供休闲的场地。当您在作战中被逐出时，您可以前往这些场地游玩。
- **出生点小屋**：每一名玩家在进入服务器之后都会出生在这里。这里是整个服务器的重要枢纽，供玩家开始游戏、加入游戏、切换地图，或前往其他场景。

## 功能类
- **物资商店**：这里提供了很多的作战物资，可为玩家在作战中的胜利助一臂之力。
- **资金**：服务器的基础货币，可以让你在物资商店中买到更高级的装备。目前资金只能通过每日签到和作战胜利获得。
- **服务器菜单**：菜单是服务器功能的集成者，它可以更方便的让你执行一系列操作，比如快速访问服务器物资商店、使用兑换码、签到等等。
