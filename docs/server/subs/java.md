# 简介
Comet Java 私服是一个由社区强力驱动，且运行在物理机上的插件模组混合服务器。

服务器装载了众多休闲娱乐模组，主要玩法为生存、养老和生电，支持手机和电脑共同游玩（手机需要安装能运行Java版MC的启动器和渲染器）。

得益于多世界插件，服务器可以可以实现将两个周目的存档同时运行，玩家可随意穿梭于两个存档的主世界和下界，并且物品始终互通。

:::tip
此服务器的一周目之前和其他子服务器运行在一台服务器上，且仅供管理员游玩。
:::

## 加入服务器
1. 打开群文件，找到对应设备类型的最新版本整合包，并下载；
2. 打开启动器（PCL、FCL等），导入整合包，开始下载模组内容；
3. 整合包安装完毕后，启动游戏，打开多人游戏，直接加入服务器。

## 注意事项
1. 进入群聊中的玩家请把自己的群昵称后面加上自己的游戏名字，否则可能会被移出群聊。
2. 第一次登录服务器，玩家需要在游戏内使用“/register”注册，注册需要设置密码，若忘记密码可通过“/email”使用邮箱重置，请不要泄露你的密码和邮箱名字。
3. 若在配置安装游戏方面有困难，可联系管理共同解决。
4. 手机玩家可使用“MobileGlues”渲染器获得更快且兼容性更好的游戏体验，具体配置方法可以上网了解。
