## 简介
Comet 枪战服务器是 Comet 群组服的附属服务器之一，同时也是 Comet 服务器的一个重要的宣传项目。目前只能通过主服跨服进入（也就是不公布端口）。该服务器由自发组建的 Comet 创作者团队（包括管理和玩家）维护与运营。

Comet 枪战服务器的主要玩法是以射击为主的团队竞技，灵感来源于“CS:GO”等游戏。

无论你是喜爱MC环境下的枪战游戏，还是想要和好友寻找一个切磋的场地，Comet 枪战服务器绝对是每名玩家的不二之选。

如果您是第一次游玩基岩版的枪战游戏并且是第一次接触基岩版枪战模组，那么这个服务器绝对会让你难以忘怀。

## 注意事项-进服前须知
由于 Comet 枪战服务器的特殊性，目前服务器的所有地图都需要至少两个人才能开始游戏，所以该服务器并不适合单人游戏。在服务器为1人时或您邀请不到人时，您可能不会在服务器玩太久。同理，一般仅仅因为枪战服而来到我们服务器的人，大多数都不会在服务器特别活跃、待的特别久，久而久之就无聊了，然后成潜水玩家了。我们推荐每一个想要来到我们枪战服务器游玩的玩家，也去我们其他子服务器体验体验，例如生存服务器、空岛服务器等，都很优质，管理处事利索，玩家氛围很好，玩家质量很高，社区化很强，插件定制化特别高，防熊服措施做的特别到位，生存服还可以获取Xbox成就。

总而言之，把 Comet 枪战服务器当成一个切磋休闲区就好了。
