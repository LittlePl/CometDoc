## 群审核方案
### 确保来历明确
在加入QQ群之前，请确保你有一个常用的，且QQ等级不低于15级的QQ账户。加入群聊时您需要提供你**从哪里得知的该服务器**，之后群管理员会审核您的进群申请，且会从多方面（例如您的QQ头像、QQ等级、QQ资料卡等）评估你的QQ号使用情况。若无特殊情况，群管理会在24小时内回应您的请求（最快3~5分钟）。

### 疑似新注册账号通常不会通过申请
对于QQ等级过低（疑似新注册账号）且个人资料几乎空白的用户，原则上管理员不会通过申请。若您实在想玩这个服务器但却在QQ账号这方面不符合我们的要求，您可以[和我们取得联系](https://doc.mccomet.top/server/group/admingroup.html)，以便于进一步核实情况。

### 与群内玩家一起核实新注册账号
如果您是被群内的玩家邀请进服务器的，且您的QQ号疑似新注册账号，我们可能会配合群内玩家提供一些证明。若证明属实，我们会自动放行该账号进入群聊。

:::warning 注意
- 使用疑似小号作出服务器违规行为，将与大号连坐处理。
- 收购一切违规账号玩家的物品、金币一律视为违规玩家的小号与同谋，给予连坐封号处理。
:::

## 进群后须知
1. 本群目前支持**群号搜索**以及**二维码加入**。

2. **本群聊会不定期清理不活跃成员。**进入QQ群之后，请务必平时保持活跃，否则可能会在某一次管理决定清理QQ群不活跃用户时被移除群聊，以至于后续游玩服务器受到影响。

3. 请各位玩家把自己的群昵称改为：`你的QQ昵称（你的游戏名）`
例：`牛马（wocenimenma）`，否则有可能会有被群管理员清除掉的潜在风险。

## 群头衔 / 群称号
1. QQ群等级超过60级的群成员可私聊群主，定制一个群头衔（作为群聊水军认证）；

2. 若您对服务器进行了赞助，可直接携带相关证明私聊服主定制群头衔；

3. 群头衔限制6个字符，因此总共不能超过6个汉字或6个字母。

## 退出群聊
1. 当群成员退出群聊时，**该玩家绑定的 XboxID 将自动从所有服务器解绑，并在群聊中公示，已退群的玩家将无法加入服务器**，除非重新加入群聊并添加到白名单，否则将被服务器的白名单拒绝连接。

2. 退出群聊后，**该群成员的QQ号将被群Bot公示**，所有群成员均可查看，以便其他认识您的好友了解您的情况。此外，**群管理员可能会向您的QQ邮箱发送慰问信，询问您退群的原因。**
