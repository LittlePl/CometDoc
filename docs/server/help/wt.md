# 进不去服务器：常见问题解答
## 卡进度条
进入服务器提示：正在下载资源包，进度条不走动，卡0.00MB。请点击加入而不是下载并加入，服务器重启后可点击下载并加入下载文件。这种情况通常出现于第一次进入服务器的玩家，可能长时间运行导致的无法下载资源包！

## 加载失败
主页登录不了账号，服务器进去提示未经过验证，是本地网络的问题，重启游戏或者重启网络试试？（飞行模式开15秒关闭切换IP，路由器拔掉15秒插回去切换IP）也有可能是你在学习知识？登录了微软账号，绑定了游戏ID，你就可以直接进入服务器。

## 地址无效
无法连接到服务器、无效IP、检查您的互联网连接并重试等问题。 排除是IP端口不正确问题，获取最新IP和端口，仍然出现的话：请尝试使用WIFI网络进入，WIFI网络可以进，说明是您没有开启“允许流量数据进行游戏”，在Minecraft主页进入设置，允许后重新进入服务器尝试。
