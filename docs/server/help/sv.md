# Comet 服务器：常见问题解答
## 服务器的IP和端口在哪里？
新人进群请先查看Q群管家给你发的信息（或者查看置顶群公告）进入新人引导，这里面就有服务器的IP、端口等信息。为了服务器的稳定运行，我们拒绝任何形式的IP端口泄露和传播，包括把IP端口直接发送至群聊或私聊至其他玩家等行为。

## 为何我们要求进群获取服务器IP和端口：
1. 进群获取配合我们的白名单系统使用，防止有人开小号，从而破坏其他玩家游戏体验；
2. 方便我们管理玩家，约束玩家游戏行为，有效防止熊服事件发生；
3. 方便我们通知玩家有关于服务器的最新动态，让玩家更好融入服务器环境，比如服务器IP端口变更、服务器更新日志和活动通知；
4. 筛选快餐式玩家。我们的服务器长期开服，我们不希望玩家一进游戏没玩多大一会儿就退游，为我们的服务器维护工作造成负担。

综上所述，我们这么做的原因，力求维护服务器良好环境，并筛选恶意玩家。虽然这么做对玩家严格了一点，但是服务器想要有一个积极向上的游戏环境，必然需要所有的玩家共同努力。
那些因此放弃的玩家，正是那些急于求成，并可能包含恶意的玩家。
实际上，你只要复制粘贴个QQ号或者扫个码进群，然后看完我们制作的信任引导就可以进游戏玩了呗。
*这很简单呀，这很难么？*

## 新人引导是怎么回事？
为了确保服务器的持续稳定运行，并保障每位玩家的游戏数据安全，我们精心设计了一套详尽的引导教程。这套教程虽然较为细致，但它旨在为服务器创造一个良好的游戏环境、规范玩家的行为，防止恶意玩家对服务器造成损害，以助力服务器长久发展。同时，我们也注意到并致力于减少游戏内容的低龄化倾向，以提供一个成熟且友好的游戏环境。

我们理解，不是所有玩家都倾向于投入大量时间来学习这个服务器的各个方面。如果您更倾向于轻松、快餐式的游戏体验，或是希望享受更为宽松的管理环境，那么我们的服务器可能不是您的首选。

然而，如果您愿意投入时间和精力，与我们一起成长和探索，我们承诺将为您提供一个充满挑战与成就的游戏旅程。选择加入我们，意味着您将获得一段难忘的游戏体验，以及一个稳定、安全且充满活力的游戏社区。我们期待您的加入，并共同创造美好的游戏时光。

## 为什么违反服务器规定没有时效？
服务器内未经允许拿别人的东西、未经允许破坏他人的建筑、未经允许拆别人的房子，都会**永久封号**，请游玩服务器时，和平友善游玩，避免不必要的纷争。
所有影响其他玩家体验感的行为，均 **永久封号**。
综上所述，因为是永久封号，才没有时效。

## 进入服务器提示我没有获邀。
这种情况通常是您没有绑定白名单或者是你绑定的账号有误。您可阅览服务器新手引导获取相关方法。

## 为什么服务器有时候版本总是跟不上官方版？
Comet 进行服务器版本更新，首先要确定以下几点需求：

- 需要插件加载器能支持最新版；
- 需要所有需要适配的插件的作者同步更新插件；
- 需要 mod 作者同步更新；
- 需要前置作者同步更新；
- 服务器管理员平时可能忙于其他事情，可以联系相关管理员寻找原因；
- 不是你一句快更新就可以解决的。

不要问为什么不更新。不更新有不更新的道理。

## 为什么你们的服务器有白名单系统？
白名单可以有效防止恶意玩家切小号开挂或刷物资，一定程度上起到约束玩家游戏行为的作用；同时它也会极大地方便我们处理服务器突发情况，可帮助服务器长久稳定运行。目前白名单系统可能对部分玩家不太方便，请大家谅解。
