# 服务器插件概览
## 插件是什么？
插件是用来增强和定制游戏服务器功能的工具。

插件可以**为服务器添加许多新功能**，比如侧边栏、称号系统、玩家交易市场等，增强游戏的可玩性和多样性。

插件可以用来**优化服务器的性能**，通过调整原生游戏的部分游戏机制，达到减少卡顿并优化服务器计算性能的目的，或用来修补游戏中存在的漏洞，为玩家提供更平稳的游戏体验。

插件也可以用来让服务器管理员更轻松地管理玩家行为、监控游戏世界等。它们也为服务器管理者提供了类似于游戏工具箱的功能，比如建造助手等。

插件也可以**提供许多方便玩家进行游戏的功能**，让游戏体验更上一层楼，比如设置传送点、玩家间传送、玩家公会、领地等。有一说一，这也是我们爱玩服务器的原因之一。
