# Comet 陆上彗星服务器
## 简介
Comet 陆上彗星服务器，是一个我的世界国际基岩版服务器，创办于2023年2月，其管理层成员均有4年以上开服经历，在2022年就有过合作。

服务器使用 BDS + LeviLamina 开设，是一个群组服务器，开设生存服务器、空岛服务器、小游戏服务器等子服务器，但我们仍然以原汁原味的原版生存服务器为主。

服务器包含众多有利于玩家生存的插件，还会加入一点特色设定（如精英怪）来丰富玩家体验。服务器也会在假期发布多个招新视频，视频内容主要以服主的视角，与其他玩家共同生存，遇到的奇特经历为主。

服务器最近还开设了Java服务器，由服务器玩家强力驱动，服务器管理员负责代理，是我们在Java版MC开服领域的一个新的尝试。

## 服务器定位
**基于插件的 <font color="#39895A">生存</font>/<font color="#3E6FDC">模组</font>/<font color="#9440C7">空岛</font>/<font color="#880708">枪战</font> 服，<font color="#39895A">混战</font>/<font color="#3E6FDC">养老</font>/<font color="#880708">休闲</font> 。**

为保障玩家的游戏体验，我们会定期检查服务器的后台行为日志：**严禁开挂和使用矿透等一切的第三方工具，禁止玩家恶意刷屏的行为**，若遇到请积极检举，我们会在最快的时间内给您一个满意的答复。
