# 服务器备份系统
## 双重保障
我们极其重视数据安全。服务器会在每天凌晨4:00对存档进行自动备份，并且我们会在稍后上传至云端。

我们还会定期地对插件数据以及其他服务器数据进行维护备份。
## 工作形式
对于自动备份，有着插件的帮助，服务器可以做到不停服热备份，将服务器存档打包进一个压缩包里。

手动备份通常涉及到插件、日志等重要数据的备份。这一过程通常由人工操作。在服务器的低谷期，我们暂时关闭服务器，然后手动将相关文件打包成压缩包，再上传到云端进行保存。
