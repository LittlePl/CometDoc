## 简介
Comet 创作者团队是一个由玩家和管理员组成的团体，致力于为 Comet 枪战服务器等娱乐服务器做出贡献，他们是服务器项目进展的中坚力量。团队成员的贡献包括但不限于为服务器制作建筑、出谋划策等。随着服务器的发展，团队成员将有机会发挥更大的潜力，而不仅限于这个服务器。
