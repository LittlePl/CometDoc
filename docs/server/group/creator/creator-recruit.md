# Comet 创作者团队招募：分享你的创意，共筑游戏梦想
如果您有着不错的想法，并渴望在充分利用服务器资源的基础上让自己创意十足的作品被更多人享受，您可申请成为 Comet 创作者团队的一员。

我们诚挚地邀请您加入 Comet 创作者团队，作为团队的一员，您将有机会与其他才华横溢的创作者们一起，共同打造一个更加精彩和多元化的游戏世界。

## 什么是 Comet 创作者团队

Comet 创作者团队是一个由玩家和管理员组成的团体，致力于为 Comet 枪战服务器做出贡献，他们是服务器项目进展的中坚力量。团队成员的贡献包括但不限于为服务器制作建筑、出谋划策等。随着服务器的发展，团队成员将有机会发挥更大的潜力，而不仅限于这个服务器。

## 加入之后
当你成为团队成员后，**你将被授予团队涉及服务器的OP权限**，可自由执行命令、切换创造模式等。

但因为玩家可以使用OP权限修改自己的余额，**我们将禁止这些玩家在这些服务器访问通用货币菜单**，防止其非法篡改其他服务器的账户。

## 如何申请

申请团队成员需要与服务器管理员 BaiYang（QQ:3124727955）或 Cheyi（QQ:17658365）协商，待审核之后我们将您拉入创作者群聊，即代表您已经申请成功，您可以在这里与其他人自由交流您的游戏想法。

您可在服务器QQ群聊的成员列表中找到他们。

## 申请要求

通常来讲，申请成为创作者需要满足以下门槛：

- 有着不错的想法，想要在这个服务器上实践；
- 自己有能力为服务器做贡献，具备一定的建筑水平、指令水平或查缺补漏的水平；
- 自觉遵守纪律，不擅自改动服务器的其他地方，不要在服务器作(zuō)得太过分以免其他人的作品毁于一旦；
- 自己有能力长期担任这一职务，承诺不建烂尾楼、不短期内跑路；
- 可以与其他人畅快合作，共同商量作品的设计方案与服务器下一步的目标；
- 承诺所拥有的OP权限仅用于贡献服务器，在进行游戏时绝不使用权限，以免破坏其他玩家的游戏体验。

一般地，老玩家申请成为创作者的门槛比新玩家低得多，因为我们更信任他们。

如果您认为自己够不着上述门槛，但仍然有自认为不错的建议，欢迎私信任意管理员，我们将按照实际情况进行采纳，感谢你的付出。
