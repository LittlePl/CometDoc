# 服务器税收组

## 简介
服务器税收组是一个从未经过管理层正式认可、也无官方明确定义的服务器 “职能组织”，该组织以 “维护服务器经济稳定”“实现公平经济平衡” 为合理名义，通过各类自定规则直接扣除玩家或联盟的游戏金币，以此达成管理员口中的 “经济调控” 目标。本质上，这只是服务器管理员用于维持所谓 “经济平衡”、缓解服务器通胀的对外名号与执行手段。

这一机制最早由服主 Kongtou 提出并主导落地。

## 组织成员
- Kongtou（服主）
:::danger 注意
服务器管理组不属于税收组成员，不参与其规则制定与执行工作。
:::

## 采取手段
该组织以 **“适配服务器经济环境”** 为依据，不定期向玩家收取费用、实施经济限制，核心手段包括但不限于：
- 玩家转账税收，通过资金流转抽取对应费用；
- 不定期依据玩家账户余额定向征税，税率根据需求灵活调整；
- 服务器全局商店税收调控，官方购买商店税率可随时上调；
- 动态调整回收商店规则，下调商品回收价、下架部分回收品类；
- 领地租金税收调整，以 “玩家反馈优化” 为由，将领地租金从 0.02% 直接上调至 0.1%；
- 高资产玩家专项征税，对存款不低于 1 万金币的个人 / 帮会，按 35% 税率征收资产税；
- 帮派定期定额征税，指定联盟每周缴纳固定高额税款；
- 推出象征性帮派返税活动，单次纳税超 1000 金币仅返还 10 金币，以 “鼓励纳税” 为名绑定帮派义务；
- 领地保护与纳税挂钩，非纳税玩家在无领地区域遭遇物品被盗、建筑被毁，管理组不予受理。

## 处罚措施
若玩家或联盟未遵守税收组规则（如逃税、逾期未缴），该组织将依托管理员权限实施处罚，包括但不限于：
- 处以高额金币罚金；
- 对违规联盟追加定期苛捐杂税；
- 逾期未缴则直接查封联盟资产。

若玩家拒绝缴纳罚金、认错态度未达要求，税收组将启动强制执行程序：
- 没收帮派内所有成员 95% 的账户金币余额；
- 没收全部可变现为金币的资产，涵盖回收商店可售物品、官方商店可交易物资；
- 玩家生电设施无条件充公；

**所有规则辩解无效，税收相关事宜最终解释权归税收组所有。**

## 特殊情况
法定节假日期间，税收组全员进入休息状态，凡涉及服务器经济、商店、税收相关的咨询与问题，均以 “税收组休息” 为由暂缓处理，统一延后至节假日结束再恢复响应，相关问题的具体解决时间由税收组自行安排。
