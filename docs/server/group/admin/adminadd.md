# Comet 陆上彗星 管理组招募计划

欢迎来到**Comet 陆上彗星**管理组招募页面！我们衷心感谢每一位愿意为服务器付出时间与热情的玩家。管理组的每一位成员都是服务器成长的重要支柱，正是因为有了大家的共同努力，Comet 才能持续提供优质的游戏体验。

本章节将详细说明**Comet 陆上彗星**管理组的组成结构、招募要求以及相关细节，帮助您全面了解如何成为我们的一员。

## 管理组构成
Comet 陆上彗星管理组分为以下五大类型，各司其职，协同合作：
- 审核管理：负责玩家申请审核、行为监督与规则执行。
- 技术管理：负责插件开发、数据包制作、系统维护与技术支援。
- 事务管理：负责玩法指导、社区沟通、活动组织与日常事务处理。
- 宣传管理：负责视频录制、内容剪辑、平台运营与社区宣传。
- 美工管理：负责服务器海报设计、宣传物料制作、视觉美化与图片处理。

## 基本申请条件
我们期待您具备以下基础素质：
1. 正确的价值观与良好的判断力；
2. 情绪稳定，善于沟通，具备团队协作精神；
3. 拥有较为充足的业余时间参与服务器管理；
4. 出于对团队成员时间与学业负责的考虑，原则上高中及以下学历的申请者暂不接纳，建议优先专注于学业；
5. 真正热爱这个服务器，并愿意为社区建设贡献力量。

## 各类别具体要求
### 审核管理
- 三观端正，做事公正，严格遵守规章制度；
- 能主动学习并熟练掌握审核流程与标准；
- 需投入大量时间进行申请处理与行为监督；
- 具备较强的责任心和细致观察力。

### 技术管理
- 本服务器为**我的世界国际基岩版**，使用 **LeviLamina 插件加载器**，插件架构包括 JS、原生（C++）、NodeJS 等；
- 插件开发方向：熟悉 JS，能独立完成基岩版插件开发、调试与维护；熟悉 LeviLamina 插件加载器与 Minecraft 基岩版各版本特性与开发规范；
- 数据包开发方向：熟悉基岩版行为包规则，能实现各类玩法与功能逻辑；
::: warning 注意
注：官网/机器人开发、设备维护等岗位暂未开放招募。
:::

### 事务管理
- 待人友善，有耐心，具备较强的沟通与协调能力；
- 熟悉服务器各项玩法、机制与社区文化；
- 能够积极协助玩家解决问题，维护良好的游戏氛围；
- 时间充裕，能及时响应社区需求。

### 宣传管理
- 掌握至少一款剪辑软件（如 Premiere、剪映等），对视频节奏、叙事有一定理解；
- 喜欢记录游戏日常与社区精彩瞬间，善于捕捉有趣内容；
- 具备良好的审美与创意，能独立完成宣传内容制作；
- 拥有持续且稳定的空闲时间。

### 美工管理
- 熟练使用 PS 等主流图形处理、修图软件；
- 具备良好审美，能独立完成海报、宣传图、视频封面、社区视觉物料设计；
- 风格贴合服务器氛围，创意清晰、出图效率稳定；
- 有相关设计作品或经验者优先。

## 服务器资金来源
- 服务器主要运维资金来自赞助，当本月赞助过少时我们会进行补贴，维持社区正常运转；
- 管理组全体成员无薪任职，均为爱发电；
- 服务器所有赞助资金由服主 **@Yanda** 设立独立账户管理，每一笔收支均公开透明；
- 结余资金将用于服务器应急需求与未来硬件升级，确保服务器长期稳定运行。

## 我们期待这样的你
如果你认同我们的理念，具备相应的能力与时间，并愿意与 Comet 陆上彗星服务器共同成长，欢迎加入我们的团队。

在这里，你不仅能积累管理经验、提升技术/设计/创作能力，还能结识一群志同道合的伙伴，共同打造更好的游戏社区。

**申请方式**：请准备好个人介绍与相关作品/经验说明，联系服主或现有管理员进行初步沟通。

感谢你的阅读与关注，期待在管理组见到你！
