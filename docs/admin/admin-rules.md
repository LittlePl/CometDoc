# 管理员守则
以下守则旨在规范管理员的行为，维护服务器的良好秩序，为玩家提供一个公平、公正、和谐的游戏环境。希望各位管理员能够严格遵守守则，认真履行职责，共同推动服务器的发展。

:::warning 约束力明示
长期参与服务器租费分摊的**服主**不受此文档的约束。
:::

## 公务与私事
作为服务器管理员，必须明确区分公务与私事，做到公私分明。在使用OP权限处理服务器事务时，应以**维护服务器正常运行、保障玩家权益**为首要目标，**不得将个人情绪、利益或私人恩怨带入其中。**例如，在处理玩家投诉或纠纷时，要秉持客观公正的态度，依据服务器规则和实际情况进行判断和处理，而不是基于个人好恶或与玩家之间的私人关系来做出决定。

## 管理员负激励
管理员若被玩家投诉，服务器主管将对被投诉管理员实行**负激励处置**，记录在专门的**负激励表**上。一旦某个管理员的负激励值达到某个阈值，该管理员将会**面临一系列的处罚**，如罚款、限制部分管理功能等，严重者会被挂起管理员权限，甚至被永久撤职。

管理员应时刻保持职业素养，避免因不当行为而受到负激励，从而影响自己在服务器中的形象和声誉。

## 避免神权行为
以下管理员行为我们定义为神权行为：
- 使用命令、创造模式等**原版作弊措施**，对玩家**实施较严重的干扰，以至于阻碍玩家正常游戏**的（如频繁tp玩家至自己、活动抢跑等）。
- 使用管理员权限或后台等工具**擅自调取玩家插件数据**，且发生**非法篡改**的。
- 使用管理员权限随意修改自己或他人的部分数据（如经济系统余额），且**干扰到服务器经济系统平衡**的。
- 和其他玩家**产生互动**的同时，**使用OP权限或后台进行了正常玩家根本无法做到的活动**（如**擅自查询日志**、持有非法物品、放置命令方块、飞行等），**对服务器的公平环境有潜在威胁的。**
- 使用管理员专属物品（命令方块、拒绝方块等）进行**私用**的。
- 向玩家或给自己**提供非法装备**。
- **擅自**调取服务器行为日志**对侵犯自己虚拟财产利益的玩家进行查找与定位**的。
- **擅自访问他人联盟私密信息，或随意在他人联盟发布公告的。**
- 因**私事私务**而动用管理员权限，对玩家进行**禁言、封禁等行为**。
- 多次使用“@全体成员”等功能进行扰民的。
- ……

服务器倡导人人平等，**请把管理员权限用在该用的地方上。**若管理员发生了以上行为或类似行为，服务器主管将会根据情况，对涉事管理员做出一定管制措施，并**给予负激励，记录在负激励表上。**

管理员应时刻牢记自己的职责和使命，尊重每一位玩家，公平公正地行使自己的权限，维护服务器的良好秩序。

## 发放福利或补偿
管理员可以在有适当理由（如服务器达到目标、被迫关闭、管理员失手对其他玩家造成损失）的情况下，依据实际情况对服务器玩家发放适当的福利或补偿，且与神权行为互不冲突。奖励不应对玩家影响太大，且所有物品遵循原版规则。

## 适当使用“@全体成员”与群待办
如果您是群管理员，您可以使用“@全体成员”和群待办功能，用来给群成员一些提醒。

“@全体成员”会导致群内的所有成员**强制收到通知**，无论他们是否开启群消息免打扰。不当使用此功能可能引起群成员厌烦，甚至造成玩家离开群聊等不可逆转的情况。建议仅用于提醒服务器公告等**与服务器有关的内容**（公务），避免使用这个功能来提醒群员一些无关紧要的内容。

群待办通常用于设置群投票、服务器问卷等，提醒的效果比@全体成员还要强烈，并始终在聊天页面上方显示。若服务器主管发现与服务器无关的内容被设置群待办，我们会立即撤除该群待办，并追溯至原管理员进行负激励处理。

## 服务器后台账号使用规范
### 绝对禁止租借转让
服务器后台账号**仅对应管理员自己能够使用**，**绝对禁止租借、转让账号到其他管理员或人员处**，若被服务器主管发现，其账号与被借管理员的对应账号均做**无限期封禁处理**，并要求涉事管理员进行**反思**。

后台账号是管理员履行职责的重要工具，具有高度的保密性和安全性，一旦租借或转让，可能会导致服务器信息泄露、管理混乱等问题，严重影响服务器的正常运行。

### 慎用文件管理功能
文件管理功能属于高危功能，**使用不当可能会影响服务器正常运行。**一般人若有需要仅用作查找文件即可。若非专业人员，**请不要擅自对服务器文件进行删除或修改。**

##  服务器行为日志调取规范
原则上，服务器行为日志仅允许被用在**处理公务**，如查找刷物玩家、外挂玩家等违反服务器管理条例的玩家，**请不要用于私事用途。**
