## 建设文档站的初衷
Comet 文档站，是 Comet 服务器的知识库，我们致力于将与服务器相关的所有信息、活动和内容汇总到这里。

无论是历史事件、游戏攻略、常见问题还是社区动态，我们都希望被详细记录。未来，所有玩家都可以随时查阅这些内容，回顾和共享属于 Comet 服务器的美好回忆。

我们希望这个文档站不仅仅是一个信息库，更是每一位玩家在 Comet 服务器冒险旅程中的珍贵见证。

## 积极参与建设文档站
正所谓“众人拾柴火焰高”。单靠管理员的力量，想要打造一本适合所有玩家使用的全面手册几乎是不可能的。Comet 文档站的建设需要玩家们的共同参与与支持，只有大家齐心协力，才能将它打造得更加完善与实用。

正如著名科学家爱因斯坦所说：“只有团队合作，才能创造出非凡的成就。”我们期待每一位玩家在 Comet 服务器的冒险旅程中，贡献自己的知识与经验，共同建设这个宝贵的知识库，让 Comet 文档站成为每个玩家值得依赖和回顾的资源中心！

## 注册文档站账户
若您想为 Comet 文档站的建设贡献一份力量，您需要有一个文档站账户。

在您注册账户之前，您应已阅读[ Comet 服务器项目章程](https://doc.mccomet.cc/docs/admin/chart " Comet 服务器项目章程")，并悉知：**注册账户仅仅是为使用者开放部分文档的编辑权限和评论功能而已，仅方便自己随时编辑，并且我不会将账号和密码分享给任何人。**

### 注册建议
- 您是游玩过服务器一段时间的老玩家，对服务器的相关内容比较了解；
- 您熟悉 MarkDown 的基本用法；
- 您具备一定的编辑能力、排版能力和信息素养；
- 您拥有平板/PC等大屏设备，用手机编辑文档会导致文档站的部分UI不可见，编辑文档将会非常吃力。

我们绝不排斥在编辑文档方面有困难但仍然有贡献想法的玩家。若您觉得自己不满足以上建议，您可以随时把您建议修改或改善的地方报告给任意管理员，管理员会酌情考虑并权衡，我们同时也会非常感谢您。

### 准备工作
**注册文档站账户的全过程需要管理员协助**。在注册账户之前，请准备好以下信息，并与任意管理员取得联系：

- **账户名称**：规定一个账户名称，用作账户唯一标识；
- **您的XboxID**：需要已绑定到服务器白名单并与联系人QQ对应；
- **您的电子邮箱地址**：注册账户必要条件。您也可以使用邮箱当作用户名登录；
- **您平时游玩服务器主要使用的设备**：建议平板电脑、PC等大屏设备；
- **您想注册账号的目的或个人想法**（比如我是服务器的忠实玩家，并且有很多不错的想法，想和大家分享）；
- **您想管理的板块**（比如我想记录服务器的地标、设施与活动，并适当参考该板块文档已有的信息去调整我的格式）；
- **其他补充内容**（提高过审率）。

将以上要求编辑成消息，并发送到管理员，管理员预计会在15个工作日内给予答复。老玩家将会优先审核。

### 账号功能
玩家有了文档站账号之后，相比于游客来说，可以做到：
- **编辑文档**：玩家可以编辑自己申请板块内文档的所有内容；
- **评论文档**：玩家可以在任意开放评论功能的版块内评论。
	- **注意**：建议在评论内留下你对这个文档的见解和建议，管理员能看到你的评论，并适当采纳。请不要随意留下内容，因为任何人都能看到你的评论（包括游客）。

### 注意
1. 作者是 **admin** 的版块，玩家不能申请管理。

2. 账号密码无需提供给管理员。管理员会给予你的账号创建初始密码，在您登录上文档站之后，您需要自主修改为你自己的密码。除了你以外的任何人都无法获取你自己设置的密码。

3. 在您保存或发布编写的文档时，您的 XboxID 就是你的作者名，会显示在文档的末尾；

4. 一般地，老玩家申请账户的门槛比新玩家低得多，因为老玩家对服务器的理解远远是比其他新玩家多得多的，并且我们更信任他们。

## 数据无价！
请务必谨慎，为了避免编辑功能被滥用，造成文档站内容被不当篡改，**绝对不要随意将 Comet 文档站的账号分享给周围的人。所有改动一旦发生，将无法撤销。**我们希望每一位使用者都能珍惜并维护这个平台的安全性与完整性，以确保文档站始终为服务器玩家提供准确可靠的信息资源。
