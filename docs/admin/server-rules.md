# 黑名单违规介绍
::: warning 请注意
加粗内容代表此行为为近期频发、且容易复现，请玩家们格外注意。若这些条目违规，将受到更严厉的惩罚。
:::

## 账号/安全相关
- 共享账号给多人同时使用，导致服务器负载异常或影响正常游戏秩序。
- 使用程序批量注册账号，占用服务器资源。
- 通过非正版游戏进入服务器。
- 通过第三方启动器进入服务器。
- 使用抓包程序进入服务器。
- 使用第三方修改端进入服务器。
- 使用非自己注册的 Xbox 账号进入服务器（黑号）。
- 利用攻击程序、网站、工具等对我方任意节点造成攻击的行为。

## 游戏内经济系统违规
- **利用漏洞刷取游戏内货币或珍贵道具，破坏经济平衡。**
- 恶意囤积稀有物品，哄抬物价，扰乱正常交易市场。
- **通过非正常手段获取物品（BUG 刷物品等）。**

## 红石与建筑设计
- 设计极其复杂且不合理的红石电路，导致服务器卡顿甚至死机，比如无限制循环的红石脉冲电路。
- 在服务器限定的建筑高度或区域范围外进行违规建造，影响服务器的地图管理和渲染。
- **在服务器内建造不良内容。**
- **创作涉嫌违法的内容。**
- 建造无意义的建筑、标志、破坏地形。
- 未经允许破坏服务器公共设施的行为。
- **在服务器内建造涉政、涉密等违法行为。**
- **在服务器内建造黄、赌、毒等违法行为。**
- **在服务器内建造邪教、信仰非法帮派等行为。**

## 插件与资源使用
- 私自修改服务器自带插件或核心文件，导致服务器功能异常。
- 上传、使用含有恶意代码或病毒的自定义资源包，危害其他玩家设备安全。
- 通过非正常方式寻找地形（含无脑刷区块跑图）。
- **利用游戏特性导致服务器崩溃。**
- 利用插件 / mod 等作弊手段进入游戏。
- **使用矿透材质包进入服务器。**
- **破坏玩家领地内建筑 / 财产。**
- **未报备做大型红石生电设施。**
- **在地图过远的坐标位置（如坐标50w+）长期活动，如定居。**
- **使用领地系统对服务器唯一自然结构（如末地的传送门）或公共设施（如出生点）进行圈地。**

## 服务器特色机制违规
- **使用骷髅刷怪笼或无限产出骷髅的设施刷骷髅。**

## 社交互动类
- 频繁发送私人消息骚扰其他玩家，影响他人游戏体验。
- 组织玩家进行恶意投票，干扰服务器正常的管理决策（如踢人投票等） 。
- **在服务器内、外刷屏、骂人、骚扰、侮辱、诋毁、引战、水军等行为。**
- **在服务器内、外宣传其他服务器、QQ 群、公众号、社区、论坛、博客、网站、App、系统等。**
- **在服务器内堵出生点、欺负刚进服不久的新人，使玩家无法正常游戏。**
- 进行诈骗、引导线下等行为。
- 造谣、传谣与事实不符的言论。
- 教唆他人犯罪、犯法等行为。
- 教唆他人违反本服务器规则等行为。
- 宣传、信仰、传播：邪教、非法帮派等。
- 发表、产生不利于社会主义核心价值观的内容。
- 传播本服务器内部文件。
- 开展网络暴力、人肉搜索、深度伪造、流量造假、操纵账号等违法活动。
- **恶意或无意泄露服务器IP端口。**

## 版本与更新违规
- 使用未经服务器认可的游戏版本或私自篡改游戏版本文件进入服务器，导致版本不兼容问题。
- 故意阻止其他玩家进行游戏版本更新，影响服务器整体的版本一致性和稳定性 。
- 因服务器版本问题引导其他玩家游玩其他服务器。*（恶意引流）*
