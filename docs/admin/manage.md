# 违规处理决定
::: info 关于
服务器准许玩家间开展抄家、玩家间 PVP 等切磋行为，但不代表这是个无政府服务器。所有玩家均需遵守正常游戏规则，严禁出现使用外挂、利用游戏漏洞等破坏公平性的违规行为。若**经核查确认违规**，我们将**根据情节严重程度采取封禁措施**。
:::

### 轻度违规
> 如宣传其他服务器、恶意崩服等。
- 对于轻度的、未对其他玩家造成实质性损害的行为，给予**7天以上、30天以下封禁处理**，并：
	- 根据实际情况**清除玩家游戏内数据**（如清空背包等）；
	- 若玩家已经过[正规渠道](./report.md "举报与申诉")申诉，且认错态度良好，我们会减轻处理。

### 中度违规
> 如漏洞刷物、黑钱、辱骂管理、不停辱骂玩家等
- 对于明显干扰服务器公平性、已干扰服务器运行，或已对其他玩家造成实质性损害的行为，经后台查证被证实的，给予**30天以上、90天以下封禁处理**，**其余同上**；

### 重度违规
> 如鉴证、摧毁公共建筑、发送包含违规内容的图片、聊天记录等
- 对于重度的、危险的、已严重危害服务器正常运行的行为，或已封禁过的玩家第二次违规，给予**永久封禁**处理，视情况上报云黑，且**不可申诉**。
