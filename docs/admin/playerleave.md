# 收购已退服玩家的资产
在 Comet 服务器中，若玩家已退群且**绑定过的游戏ID被移出白名单**，其他玩家可以选择**收购他的物品**。

## 可收购列表
以下列出了收购方可选择收购或操作退群玩家资产的清单：
### Comet 生存服务器

| 资产类型                               | 参考价（金币） |
| -------------------------------------- | -------------- |
| 玩家背包（37格+盔甲栏）                | 2000           |
| 玩家末影箱                             | 2000           |
| 玩家Home点                             | 2000           |
| 玩家联盟传送锚点（若目标玩家是盟主）   | 10000          |
| 顶替玩家联盟身份（若联盟中只有他自己） | 31000          |

### Comet 空岛服务器
| 资产类型                | 参考价（金币） |
| ----------------------- | -------------- |
| 玩家背包（37格+盔甲栏） | 1000           |
| 玩家末影箱              | 500            |
| 玩家Home点              | 400            |
| 玩家空岛区域共享权限    | 5000           |
| 玩家空岛区域永久所有权  | 20000          |

## 开始收购
玩家需**联系任意管理员**，并提供以下信息：
- 退群玩家ID（可以在群聊中找到）
- 需要的资产（见上表）

之后在游戏内向管理员转账即可，**您索要的物品或信息将全部归您所有。**

## 注意事项
1. 管理员会事先检查该玩家的资产情况，若玩家购买的是物品，且目标玩家空无一物，管理员会结束此次交易。

2. 如果购买的是物品，管理员会将物品用潜影盒或箱子打包给你；如果是Home点等信息的话，管理员会将这些信息**直接转移到您的账户下**，且无视最大限制。**注：一经售出，概不退换。**

3. 玩家资产**先到先得**，若玩家的某项资产已售出，该玩家对应的资产（如背包物品、Home点等）将被清空。

4. 在收购联盟资产时，若您**已有一个联盟**，您只能**放弃或转让自己的联盟**然后再顶替身份；若您是**其他联盟的成员**，您只能**退出您所在的联盟**然后再顶替身份；若您选择**只收购传送锚点**，目标联盟的所有传送锚点将**全部迁移至您所在的联盟**，且无视最大限制。

5. 在收购岛屿资产时，若您购买的是**共享权限**，您将加入其岛屿的共享名单，但若岛主回归，岛主可以随时将您从共享名单中删除；若购买的是**所有权**，对应玩家的空岛区域将**永久属于您**，原岛主也将**失去他的岛**，但您无法更改该区域的玩家权限。

## 验证玩家是否可被收购
我们通过以下措施来验证目标玩家是否可被收购，无论玩家是否退群，只要满足以下条件均可被收购。
- 玩家进入过对应服务器至少一次；
- 玩家已不在服务器白名单；
- 目标玩家的XUID与服务器记录的相符。

## 其他必要说明
1. 对于已退群的玩家，其所拥有的资产默认被视为**无主处理**。若您非常介意这个措施，请在离开服务器之前把您的个人物资处理好。

2. 若已退群的玩家回归，且玩家的物品已经被其他人购买，则该玩家此时背包中**不会有任何之前的物品**，也不会得到任何通知。
