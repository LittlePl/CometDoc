## 服务器定位
**基于插件的 <font color="#39895A">生存</font>/<font color="#3E6FDC">模组</font>/<font color="#9440C7">空岛</font>/<font color="#880708">枪战</font> 服，<font color="#39895A">混战</font>/<font color="#3E6FDC">养老</font>/<font color="#880708">休闲</font> 。**

为保障玩家的游戏体验，我们会定期检查服务器的后台行为日志：**严禁开挂和使用矿透等一切的第三方工具，禁止玩家恶意刷屏的行为**，若遇到请积极检举，我们会在最快的时间内给您一个满意的答复。

## 游玩准则
**玩家在服务器内的所有行为均为个人自主行为，不代表服务器立场。**所有发生的事件仅限于 Minecraft（我的世界）游戏内，**不涉及任何现实世界的事件。**

所有玩家必须遵守此准则，建筑、使用服务器功能、编辑联盟名字、发言等行为不得涉及或暗示任何现实事件。违反者将受到永久封禁的惩罚。

## 游玩须知
- **服务器存在竞争、互相争夺的玩法**，玩家互相出现资源竞争的情况是可能的，**并且我们不会过多干涉玩家的行为**，只要不崩服，不开挂其他的我们一概不会过多过问。
- **管理组只负责玩法实现，实际功能是插件提供的**，因此难免存在bug，若您有解决的意愿请在群组中反馈，但借此出现攻击、谩骂管理员等行为将会被踢出服务器。
- **游玩服务器时，请以本文档和群组公告的信息为准。**管理员的任何建议或解答仅供参考，不保证其准确性。对于过于刁钻或无意义的问题，恕不予解答。
- 如您希望对该文档作出补充，可将想要补充或修正的内容，备注信息发送至任意管理员处（可按照此排版格式编辑 Markdown 文件）。
- **Comet 陆上彗星服务器不包容任何形式的人身攻击、辱骂、恶意、违法行为**，若您在游玩过程中遇到此类行为，请及时[联系管理员](/admin/handling "举报与申诉")，被封禁玩家没有申诉机会。
- **我们保留踢出、封禁、删除玩家数据的权利**，若您不同意此条款请勿加入该服务器。

## 与管理组沟通
**特别声明：Comet 陆上彗星服务器不会对任何玩家的任何行为做出任何承诺**，包括但不限于：**解封、赔偿、回档**等。

此处不提供直接联系管理员的方式。遇到问题时，建议前往更专业、更高效的平台进行咨询。请查阅[举报与申诉](/admin/report "举报与申诉")获取更多信息，我们会在适当时间内回复你的申诉，任何私信管理员等行为将会被忽略。

## 行为规范
若您在游玩过程中遇到任何问题，请及时反馈，我们会尽力解决。

关于煽动、情绪化、**恶劣行为**（如堵出生点、逼迫别人做某事等影响玩家游戏体验感的行为），我们会直接封禁。

服务器也严厉禁止**刷物品、矿透等严重危害玩家平衡的行为**，违者也将直接封禁。

您可以查看[黑名单违规介绍](/admin/server-rules "黑名单违规介绍")以了解更多服务器违规行为。

若出现违法行为，服务器会及时保留证据并配合相关部门进行调查。

## 特殊内容
在此服务器中，由于服务器性能有限，无法做到人数与性能兼顾，因此：
- 玩家在**设计大型红石生电设施时必须向管理员报备**，发现未报备者将没收财产+拉黑处理。
- 严禁跑出主世界50w坐标，下界15w开外的地方生存，过远距离的加载会大幅度影响服务器流畅度。如有跨越主世界50万底线，地狱15万底线者第一次警告，第二次直接永久封号。

由于此服务器的特殊设定，服务器也**禁止玩家使用骷髅刷怪笼**，违者同上。

## 财产损失处理及管理员回应说明
玩家在游玩此服务器时可能会遇到财产损失。对于因**服务器原因**造成的损失，例如：
- 服务器回档、炸档；
- 因服务器公共建筑遭受严重损失而回档；
- 因服务器技术问题或不适当维护而造成的玩家背包回档等玩家财产损失；
- 因管理员行为不当而造成的玩家财产损失；

本着对玩家个人数据负责的态度，我们会采取适当措施进行补偿。但如果是基于**玩家个人原因或玩家间的行为**，例如：

- 物品被偷取；
- 据点被人为破坏等；

鉴于管理员的时间和精力限制、资源限制、操作复杂性、游戏规则的约束，以及服务器公平性，**我们将不予处理。**因此，请务必妥善保管好自己的物品，并充分利用服务器功能以确保安全。祝你在服务器中取得丰硕的成果，闯出属于自己的一片天地！
