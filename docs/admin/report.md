## 注意事项
- 您提交的所有信息，必须保持真实有效、内容完整正确，建议您多次审核自己填写的内容，避免因小失误造成无法正确处理您的问题。

- 您提交的个人信息，我方高管员工可通过该产品来访问您的数据。我们会确保采集信息的正确实用性和保护用户隐私的责任，确保您的隐私安全。若因您存在虚假举报、内容不实、造谣传谣等行为，我们有权利通过您预留的手机号和身份证对您进行起诉。

- **恶意投诉和申诉，将永久封禁无法解封。**

## 如何反馈问题到邮箱
1. 打开[举报与申诉](report.md "举报与申诉")页面。
2. 打开或下载[QQ邮箱](https://wx.mail.qq.com/ "QQ邮箱")。
3. 添加账号，可以直接选择 QQ邮箱 然后授权QQ或者微信登录即可创建/登录邮箱。（后续根据引导登录即可）
4. 按照[举报与申诉](report.md "举报与申诉")页面中要求的格式，填写正确信息，并且在左下角上传附件。核对完毕信息是否完整、有效、正确，然后点击右上方的“发送”按钮。
5. 等待处理。
