# Special Week（特别周）
## 简介
Special Week（特别周）是 Comet 生存服务器推出的周期性特色福利活动，自 2025 年暑期首次开启，后续也在不同时段持续更新迭代。

该活动围绕服务器生存玩法核心，结合不同主题为玩家打造多样化的福利体验与趣味挑战，每期特别周均会推出专属的福利政策、玩法调整或奖励机制，涵盖金币收益、功能折扣、buff 加持、物品回收、死亡规则变更等多维度内容，既丰富了生存探索的乐趣，也为不同阶段的玩家提供了差异化的游戏体验，是服务器为提升玩家游戏体验、增强社区互动推出的特色品牌活动。
