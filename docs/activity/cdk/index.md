# 兑换码（CDKey）
## 兑换码是什么?
兑换码（又称CDKey）严格来说是一串由多个字符组成的代码，玩家可以通过这串代码在游戏内兑换神秘奖励。

在 Comet 服务器中，我们发放兑换码的形式非常创新。我们大多数时候会以题目的方式来拟兑换码，通过让玩家做题、解题并按照要求来得到兑换码，玩家在得到兑换码后可进入游戏尝试领取神秘奖励。

我们采用这种设计，皆在激励玩家积极参与，并增加活动的互动性和吸引力。通过这种方式得来的兑换码，想必玩家也一定感到很有成就感，同时，考虑到其他玩家也在同样努力解题获取兑换码，这种互动也能让玩家体会到“知识就是力量”这句话的含义。

## 发放形式
我们发放兑换码的形式非常创新。我们大多数时候会以**题目**的方式来拟兑换码，通过让玩家**做题**、**解题**并**按照要求**来得到兑换码，玩家在得到兑换码后可进入游戏尝试领取神秘奖励。

## 如何发放？
我们会**不定期**地在QQ群内**以图片的形式**发放兑换码，并备注是第几期。我们通常以`群公告`、`群帖子`的形式公布兑换码，玩家可以轻而易举地找到。有时我们发现，发放已有一段时间的兑换码领取人数并不达预期，我们也会或通过`@全体成员`、`游戏内通知`等强提醒方式来提醒大家领取。

## 发放机制
众所周知，兑换码的发放需要一定的领取限制，否则容易导致兑换码被滥用或快速失效，从而影响到活动的公平性和玩家的体验。

兑换码发放支持**按时间（有效期）**、**按最大领取次数**来约束玩家领取。通常我们发放的需要玩家解题的兑换码都是按最大领取次数来限制的，具体限制方式可以看每期兑换码图片下方的注释。

通过设定限制，我们可以确保每位玩家都有公平的机会获得奖励，并且能够控制活动的节奏。

## 子服务器互通性
各个服务器的兑换码数据是不互通的，各服务器的兑换情况和玩家所得物品在服务器之间相互独立，这意味着一名玩家可领取多个服务器的奖励。我们每一期发放的兑换码都会标注适用于哪些服务器，这样玩家就能快速地判断出这期兑换码在哪个服务器发放，以及奖励能在哪个服务器使用。

单独一期的兑换码，若发放至多个服务器，其奖励也一定相同，不受影响。但同时也因为互通性，领取名额或期限会宽松很多。

## 如何领取？
当你有了可领取的兑换码之后，只需即刻启动游戏、加入**对应的**服务器，然后打开服务器菜单，在首页找到`CDKey兑换`按钮，输入您的兑换码即可领取。
