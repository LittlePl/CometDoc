# 玩家贡献兑换码
## 简介
很久之前，Comet 兑换码支持玩家参与，有意向的玩家可主动联系管理员共同编写兑换码的题目，并且奖励由玩家决定，管理员来权衡。当一切都准备完成后，管理员将与玩家确认，并以群公告的形式代发该兑换码。

:::danger 注意
由于该活动玩家的重视程度不高，并且也需要管理员付出大量的成本（例如与玩家沟通），**我们现已停止宣传并取消该项目的受理**。
:::

## 定制模板
```
● 使用方式（单选）：
① 每个玩家都能使用（无日期或次数限制）；
② 指定使用次数（仍然是每个玩家只能领取一次）；
③ 只能使用一次；
④ 指定玩家使用。

● 指定截止日期（过了该日期后玩家将不能领取该兑换码）（可设为永久有效）。

● 奖励（可多选）：
① 一个物品；
② money；
③ 一个称号：
- 可指定等级（1级~5级）；
- 可带buff；
- 可指定称号颜色；
- 可设定称号价值；
- 可指定玩家是否能删除；
- 可指定玩家是否能用它卖钱（利用了称号价值）；
- 可设置有效时长（过了这个时长后玩家将不能佩戴此称号）；

▲ 可随时联系我停止发放该CDK。
▲ 本次CDK定制不收取您游戏帐户的任何资源（余额或物品）。
▲ 奖励可任意定制并且本人可领取，但要有个度。
```
