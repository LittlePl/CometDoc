# 机器人的更多功能

## 查询指定服务器的在线人数
在`list`前加上对应服务器的名称或简称（如`生存list` `空岛服list` `枪战服务器list`）来获取指定服务器的在线人数情况。

## 查云黑
发送`查云黑 <XboxID>`可查询该玩家是否在 BlackBe 云端黑名单中。

## QQ号XboxID互查
使用命令`查qq`可查询该QQ号绑定的XboxID，支持使用`at`也支持输入QQ号；使用命令`查玩家 <XboxID>`可查询该XboxID对应的QQ，还可显示QQ头像。

## 执行游戏命令
:::tip
触发此功能需要用户权限为管理员。
:::
发送`执行<命令>` `/<命令>`可向生存服务器以后台身份执行命令并在群聊中获取执行结果。

## 新人进群人机验证
新人进群后机器人会提醒该用户在群聊中发送一串代码（验证码），若12小时后该用户仍未发送，则将该用户移出群聊。

## 查询其他子服务器状态
在“服务器状态”命令的基础上，加上对应服务器的简称（如`空岛服务器状态`）即可查询对应服务器。

## 注意事项
1. 机器人群内执行命令（如发送"/<命令>"）只有特殊人员可用，若普通玩家使用，将返回错误。

2. 在使用QQ号XboxID互查时，在提供参数时需与指令间隔一个空格（包括at也一样），否则将执行错误。

3. 在进行人机验证时，如果用户没有发送验证码，但却发送了绑定命令，系统可以通过识别该命令直接跳过人机验证，而无需用户再发送验证码。
