## 简介

拍卖系统是 **Comet 生存服务器** 的一项插件功能，旨在为玩家提供一个便捷、公正的物品交易平台。

通过该系统，玩家可以**轻松地拍卖自己不需要的物品或购买其他玩家的商品**，无需与他人直接面对面交易，从而节省时间并增加游戏互动的乐趣。

此外，拍卖系统还设有**起拍价**、**拍卖时间**、**最小加价**等设置。无论是稀有物品还是普通资源，都可以通过拍卖系统找到合适的买家或卖家，极大提升了游戏的经济体系和玩家之间的互动。

## 使用方法
可以通过以下方法访问该系统。
- **服务器菜单**：`指南针`点地或使用`/m`，在**主页**可以轻松找到该按钮。
- **指令**：聊天栏输入`/auction`并回车。

## 使用指导
当玩家想要拍卖物品时，仅需打开拍卖系统，点击`新建拍卖`，之后编辑一系列信息，然后点击确认即可。

确认后，**系统将收取与玩家起拍价等价的手续费**，若余额不够则无法上架。物品必须先在玩家的背包中才能被选择。其他物品属性，包括**起拍价、描述、拍卖时间、最小加价**，均可由玩家自定义。

注意，**物品数量默认选取整组。**例如选择了一个有32面包的槽位，将会拍卖32个面包，而不是一个面包。

当服务器有物品被拍卖时，服务器会**给所有在线玩家发送一段提示，告诉玩家什么物品正在被谁拍卖**，玩家此时可以**访问拍卖系统**参与进拍卖当中，并**提出自己的竞拍价（出价）**。

当竞拍价提出成功，**出价的玩家需支付相应的手续费**，同时物品的价格会变为该玩家的出价，并以**当前价**显示。

当拍卖时间结束后，出价最高者会得到此物品，同时起拍玩家会得到对应的价钱并通知全部在线玩家，一轮拍卖到此结束。

## 注意事项
1. 玩家每次出价不能低于拍卖玩家设置的**出价最低加价**和**物品的实时价格（当前价）**。

2. **参与竞拍的玩家若最终没有得到物品，会返还相应的手续费。**

3. 若拍卖时间结束时没有任何人出价，物品将直接返还至拍卖者处，同时返还手续费。

4. **当有人对某个物品成功出价时，该物品的拍卖时间会直接设置为15秒**。因此当看到有其他玩家对自己心仪的物品出价时，请尽快做出抉择。

5. 一个玩家只能同时创建一个拍卖。

6. **最小加价设置默认是100**，这意味着拍卖系统适合出售比较昂贵、且有价值的物品。

7. 当你为拍卖品编写较长描述时，建议在完成后把这段话复制下来，以防其他物品属性不符合要求导致上架失败，从而造成已输入的内容丢失。
