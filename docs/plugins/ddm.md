# 死亡掉落经济机制 <Badge type="tip" text="生存"/> <Badge type="tip" text="模组"/>
## 简介
死亡掉落金币机制目前已在 **Comet 生存服务器** 和 **Comet 模组服务器** 实行，玩家在死亡后将掉落一定数量的金币。

## 掉落形式
- Comet 生存服务器：
	- 以**物品（金粒）**的形式掉落。任何玩家可捡起掉落的金币并存入自己的账户。
	- 玩家手持该物品点击方块或右键单击，即可存入自己的腰包。
- Comet 模组服务器：
	- 以**坐标记录**的形式记录玩家死亡点坐标，玩家返回死亡点附近即可快速拾回掉落的金币，其他人无法获知也无法拾取。

## 掉落机制
### Comet 生存服务器
:::warning 注意
金币掉落数量上限限制在 2000 。
:::
:::danger 特例
玩家在下界维度死亡会掉落身上的全部金币，无论死法。
:::
| 攻击类型     | 攻击者   | 掉落方式         | 掉落范围       | 损失率   |
|--------------|----------|------------------|----------------|----------|
| 实体攻击     | 玩家     | 百分比掉落       | 4%~40%         | 无损失   |
|              | 其他     | 区间随机掉落     | 20~200         | 10%      |
| 实体爆炸     | 苦力怕   | 区间随机掉落     | 100~400        | 50%      |
|              | 其他     | 区间随机掉落     | 100~200        | 50%      |
| 方块爆炸     | TNT      | 百分比掉落       | 60%~100%       | 90%      |
|              | 其他     | 区间随机掉落     | 400~1000       | 90%      |
| 弹射物攻击   | 玩家     | 百分比掉落       | 10%~30%        | 10%      |
|              | 其他     | 区间随机掉落     | 10~75          | 20%      |
| 摔落         | -        | 百分比掉落       | 5%~15%         | 30%      |
| 其他形式     | -        | 区间随机掉落     | 10~200         | 2%       |

### Comet 模组服务器
死了全部掉落，回死亡点最多捡起掉落金币数的95%，最少捡起掉落金币数的65%，概率随机。

## 注意事项
1. 对于 Comet 模组服务器的掉落机制，当玩家第二次死亡导致掉落了经济，但之前未捡回第一次死亡掉落的经济，则先前掉落的经济会消失。
