# 管理员菜单
## 简介
管理员菜单，顾名思义，就是只有管理员才能使用的服务器菜单。管理员菜单集成了只有管理员才能使用的一些服务器功能，这些功能大多是方便管理员进行服务器管理的，当然也有一些特权功能，比如快速给予自己buff，以及完全隐身等。

在服务器中，普通玩家（权限为成员及以下）是无法打开管理员菜单的，只能在菜单首页中看到管理员菜单按钮。

管理员菜单是作为服务器菜单的子页存在的。也就是说管理员也需要先执行正常的指令（`/m`或工具点地），然后通过菜单主页访问管理员菜单。
