# 移动光源 <Badge type="tip" text="生存"/>
## 简介
移动光源功能目前已在 **Comet 生存服务器**实装。玩家手持支持移动光源的物品或方块，即可照亮周围。

## 指令
- `/light on|off` ：设置开启/关闭动态光源(玩家可用)

## 物品可转移至副手
**手持支持转移至副手的移动光源**，**潜行**并**右击**（放置方块），即可将该物品转移至副手。拿在副手上的移动光源可正常提供移动光源效果，且会正常占用玩家的副手槽位。
