# 定位玩家 <Badge type="tip" text="生存"/>
## 简介
寻仇系统（又称玩家追踪、查找玩家）是 **Comet 生存服务器** 的一项插件功能，可以帮助玩家定位你在线的仇家的位置，方便复仇。被定位的玩家会收到提示。
:::tip
服务器默认开启定位条，因此大多数时候无需使用该功能。
:::

## 使用方法
可以通过以下方法访问该系统。
- **服务器菜单**：`指南针`点地或使用`/m`，通过**主页→工具箱**可以找到该按钮。
- **指令**：聊天栏输入`/findplayer`并回车。

不要998，更不要888，只要**9999**，你就能**知道一指定在线玩家的当前坐标**。

## 注意事项
- 只能对当前的在线玩家进行定位。
- 定位后，该玩家当时的坐标会发送给使用者，同时扣除使用者相应的余额。
- 不能对自己进行定位。
