# 服务器菜单 <Badge type="tip" text="生存"/> <Badge type="tip" text="空岛"/> <Badge type="tip" text="枪战"/>
## 简介
服务器菜单可以极大地方便玩家使用服务器提供的一系列功能（如 /tpa 等），而不需要使用繁琐的指令。

在游戏内手持服务器的规定物品点地，即可迅速便捷地打开服务器菜单。当物品栏中没有规定物品时，玩家也可以在聊天框中输入`/m`来打开服务器菜单。

## 规定的物品
上文提到，服务器菜单可以通过规定的物品打开。在 Comet 群组服务器中，不同服务器用来打开服务器菜单的物品可能不同。以下是各个子服务器打开服务器菜单规定的物品：

- Comet 生存服务器：`追溯指针`
- Comet 空岛服务器：`时钟`
- Comet 枪战服务器：`时钟`

在大多数服务器中，玩家可以手持时钟点地来打开菜单，这些物品通常是由服务器直接分发，并且和玩家物品栏绑定，玩家无法丢弃。

而在 Comet 生存服务器中，玩家需要使用指南针来打开菜单，并且服务器不会负责分发。之所以这么做的原因是，为了迎合生存的玩法，物品不会强制绑定到玩家背包，而且指南针的合成成本较低。一切都需要玩家自己后期制作，这不仅更加有利于生存，也照顾了很多玩家背包空间管理的需求。
