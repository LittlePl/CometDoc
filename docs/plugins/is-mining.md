# 刷矿机 <Badge type="tip" text="空岛"/>
## 简介
刷矿机的原理和**刷石机**一样，当岩浆遇水生成圆石的时候，**它有概率生成其他方块，而不只是圆石。**该设定仅在 Comet 空岛服务器实装。

## 概率表
此服务器的刷矿机使用**权重**计算每个方块出现的概率，并使用二分查找（加权随机算法）优化权重选择，预先计算累积权重，使用二分查找快速定位，确保概率值有效并正确分布。以下是刷矿机可能出现方块的权重表。
