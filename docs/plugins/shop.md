# 商店系统 <Badge type="tip" text="生存"/>
## 简介
商店系统是 **Comet 生存服务器** 的一个插件功能，玩家可以通过该插件创建、管理和浏览商店，并进行物品交易，支持自动获取物品的图标，支持官方商店出售回收，让交易更加生动。

此插件原名为 ShoppingMall ，定义为“综合购物中心”插件，皆在为玩家提供自主开店、交易、广告等丰富的功能。该插件在以下几个方面提供了扩展：
- **店铺系统**：玩家可创建自己的店铺，上架物品进行售卖。
- **官方商店**：服务器管方可配置一个官方商店，用于统一上架出售物品或回收玩家的物品。
- **广告位**：支持购买广告位，将店铺置于列表前方，以提升曝光率。
- **排行榜**：展示全服店铺的排名情况（基于店铺总收益）。
- **插件管理**：OP 或管理员可通过 GUI 对市场各部分进行管理，包括广告位、不可上架物品等。

## 访问方式
- 全局指令：输入`/shop`以打开商店系统主菜单。
- 服务器菜单：`指南针`点地或使用`/m`，在**主页**可以轻松找到该按钮。

## 功能概要与介绍
### 玩家商店功能
#### 创建店铺
1. 打开商店系统主菜单；
2. 选择"我的店铺"；
3. 输入店铺名称（最多10个字符）；
4. 支付创建费用（由服务器管理员设定）。

#### 上架物品
1. 进入"我的店铺"；
2. 选择"编辑店铺物品"；
3. 选择"上架物品"；
4. 选择要上架的物品、设定价格和数量；
5. 可选择添加备注信息（最多50字符）。

#### 店铺管理
在"我的店铺 > 管理店铺"中可进行以下操作：
- **修改店铺名称**：每次修改需支付费用，有冷却时间限制。
- **设置店铺公告**：显示在店铺首页，最多100字符。
- **开启/关闭商店**：暂时不对外营业。
- **删除店铺**：需先下架所有物品。

#### 编辑商品
在"我的店铺 > 编辑店铺物品 > 编辑物品"中可进行：
- **补充库存**：从背包中补充物品。
- **修改价格**：调整物品售价。
- **修改备注**：更新物品描述。
- **下架物品**：从店铺移除并返还物品。

#### 设置折扣
1. 进入"我的店铺 > 编辑店铺物品 > 设置折扣"；
2. 选择要打折的物品；
3. 设置折扣率（1-9折）和持续时间（分钟）；
4. 折扣期间物品以打折价格出售，到期自动恢复原价。

### 购物功能
#### 浏览商店
1. 使用 `/smsc` 或通过主菜单进入购物中心；
2. 浏览列表或搜索店铺/店主；
3. 点击店铺进入查看商品。

#### 购买物品
1. 在店铺内选择想要购买的物品；
2. 输入购买数量；
3. 确认后自动扣除费用并获得物品。

#### 全局搜索
1. 使用 `/smss` 或通过主菜单进入全局搜索；
2. 可选择搜索物品或店铺/店主；
3. 输入关键词搜索；
4. 结果按价格从低到高排序。

### 官方商店
#### 购买物品
1. 通过 `/smgf` 或主菜单进入官方商店；
2. 选择"购买"；
3. 浏览或搜索物品；
4. 选择物品并确认购买。

#### 回收物品（出售）
1. 进入官方商店；
2. 选择"回收"；
3. 选择要出售的物品；
4. 输入出售数量；
5. 确认后获得相应的经济。

### 店铺图标设置
#### 图标类型
**重要说明：只有URL图标才是真正意义上的自定义图标，其他类型实际上是引用 Minecraft 本地资源。**

1. **本地图标**：
    ◦ 使用服务器插件目录中准备好的图片；
    ◦ 路径：./plugins/ShoppingMall/ui/*.png；
    ◦ 实际引用格式为：textures/ui/图片名称。

2. **URL图标**（真正的自定义图标）：
    ◦ 使用网络图片链接；
    ◦ 格式必须为 http:// 或 https:// 开头；
    ◦ 管理员可能设置域名限制。

3. **自定义路径图标**：
    ◦ 引用Minecraft内置的资源路径；
    ◦ 例如：textures/ui/store_home_icon。

#### 图标修改说明
- 每种图标类型都有独立的修改费用和冷却时间；
- 需在"我的店铺 > 修改店铺图标"中设置；
- 费用、冷却时间和功能开关由管理员控制。

#### 更多内容与建议
可前往[个性化图标](https://doc.mccomet.cc/docs/plugins/icon "个性化图标")查看。

### 广告位系统
#### 购买广告位
1. 进入"我的店铺 > 购买广告位"；
2. 选择可用的广告位槽位；
3. 确认支付费用；
4. 广告位持续固定时间后自动过期。

#### 广告效果
拥有广告位的店铺会：
- 在购物中心列表顶部显示；
- 加上"(推荐)"标识；
- 提高店铺曝光度。

### SM（ShoppingMall）大厅（聊天功能）
#### 使用聊天
1. 通过主菜单进入"SM大厅"；
2. 查看历史消息；
3. 输入消息发送到大厅。

#### @提醒功能
- 在消息中使用@玩家名 可以提醒特定玩家；
- 被@的玩家会收到提醒；
- 主菜单会显示未读@消息数量。

### 排行榜系统
- 通过主菜单进入"店铺排行榜"；
- 按总收入排序显示前20名店铺；
- 显示店铺名称、店主、总收入、销量等信息。

### 交易记录
#### 全服交易记录
- 主菜单选择"交易动态信息"；
- 查看全服最近30条交易记录。

#### 个人交易记录
在"我的店铺 > 查看交易记录"中可查看：
- 我的购买记录
- 我的售出记录

## 常见问题排查
### 配置文件问题
如果插件功能不正常，可能是因为：
- 配置文件版本较旧，缺少新功能配置项。
- 解决方法：删除config.json文件，让插件重新生成最新配置。

### 图标不显示
图标不显示或使用默认图标的原因：
- 本地图标：文件不存在或路径错误。
- URL图标：网络问题或链接失效。
- 自定义路径：Minecraft内置资源路径错误。

### 无法创建店铺
可能原因：
- 经济不足，无法支付创建费用。
- 权限不足。

### URL图标限制
管理员可能设置了域名限制，确保您的图片URL包含允许的域名。

## 经营建议
1. **定期更新商品价格**：根据市场需求调整价格。

2. **使用折扣**：吸引顾客并加速销售。
3. **撰写清晰的物品描述**：帮助买家了解商品特点。
4. **设置店铺公告**：告知顾客特别信息或促销活动。
5. **保持店铺物品多样性**：吸引不同需求的顾客。
6. **购买广告位**：提高店铺曝光度。
7. **使用SM大厅**：宣传您的店铺和特价商品。
